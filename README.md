# SOC Analyst Security Dashboard

## Project Overview

This project demonstrates the creation of a SOC Analyst Security Dashboard using Splunk Enterprise to monitor security events, detect suspicious activity, and improve incident response visibility.

The dashboard focuses on:

* Failed login attempts
* Suspicious IP activity
* Intrusion detection events
* Security monitoring and threat investigation

This project was created to strengthen hands-on SIEM experience, improve Splunk skills, and maintain cybersecurity analyst readiness for future SOC roles.

## Tools Used

* Splunk Enterprise
* SIEM Monitoring
* Cybersecurity Intrusion Dataset
* Threat Detection Analysis
* Security Event Investigation

## Dashboard Panels

### 1. Failed Login Attempts

This panel tracks failed login attempts across the environment to identify brute force attacks, password spraying, credential stuffing, and suspicious authentication behavior.

### 2. Total Failed Logins

This single-value panel showed a total of 14,475 failed login attempts within 24 hours, representing a major security concern that would require immediate SOC investigation.

### 3. Suspicious IP Activity

This panel monitors IP reputation scores to identify potentially malicious or high-risk connections that may require priority investigation.

### 4. Intrusion Detection Events

This panel tracks attack detection events and helps identify malicious activity requiring rapid analyst response.

## Key Findings

* 14,475 failed login attempts detected
* High IP reputation scores identified suspicious network activity
* 4,264 attack-detected events were found
* 5,273 normal events were also recorded

These findings simulate real-world SOC monitoring and demonstrate how analysts prioritize investigations and respond to potential threats.

## Goal of This Project

The goal of this project was to maintain and strengthen hands-on cybersecurity skills outside of formal training by building practical SIEM monitoring experience that can be demonstrated during interviews and professional discussions.
